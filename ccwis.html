Redesigning America's CCWIS

Spearheading the redesign of the front-end of mCase, a Comprehensive Child Welfare Information System (CCWIS) software solution used daily by thousands of social workers across North America.

It was time for a major UX investment in RedMane's CCWIS (Comprehensive Child Welfare Information System) product. The ask:

  • Design a new, customizable homepage that offers users quick access to their most common tasks
  • Redesign the most essential page of the application (on which users view and edit information related to a case)
  • And design the UX of new search features and interaction modalities that offer faster ways of completing key workflows

I accomplished this by iteratively prototyping solutions to user needs and validating ideas via stakeholder input and usability tests. Simultaneously I developed a design system based off the USWDS (U.S. Web Design System) but customized it to add brand, beauty, and enhancements to better suit user needs. The design system fully leverages a popular front-end library in order to make the CSS as lean and systemitized as possible, while paving the way for much faster, more organized, and more maintainable front-end development.

I'm pleased to state that the CCWIS solution in question has, since our UX revamp, outcompeted Salesforce, Microsoft, Deloitte, and others to be selected as the state-wide child welfare software solution of Mississippi and Arkansas.
